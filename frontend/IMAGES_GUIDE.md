# Руководство по замене изображений

Это временное решение для прототипа. После интеграции со Strapi все изображения будут загружаться через CMS.

## Быстрый старт

1. **Поместите изображения** в соответствующие папки:
   - `public/images/hero/` - главный экран
   - `public/images/projects/` - обложки проектов
   - `public/images/apartments/` - планировки квартир
   - `public/images/news/` - изображения новостей
   - `public/images/gallery/` - галерея проекта

2. **Обновите пути** в следующих файлах:
   - `src/lib/mockData.ts` - проекты и квартиры
   - `src/components/sections/Hero.tsx` - главный экран
   - `src/components/sections/ProjectGallery.tsx` - галерея
   - `src/components/sections/NewsSection.tsx` - новости

3. **Перезапустите сервер:**
   ```bash
   npm run dev
   ```

## Структура директорий

```
public/images/
├── hero/              # Изображения для Hero секции
├── projects/          # Обложки проектов (ЖК)
├── apartments/        # Планировки квартир
├── news/              # Изображения для новостей
└── gallery/           # Галерея проекта
```

## Важно

- Все пути к изображениям должны начинаться с `/images/...`
- Next.js автоматически обслуживает файлы из папки `public/`
- После интеграции со Strapi эти локальные файлы можно будет удалить
